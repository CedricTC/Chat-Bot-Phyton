# Chat Bot

Bu Python projesi, kullanıcıların bir sohbet botu ile etkileşimde bulunmalarını sağlar. Sohbet botu, belirli sorulara cevap verir ve yeni soruları öğrenir.

## Proje Amaçları:

- Kullanıcıların sohbet botu ile iletişim kurmasını sağlar.
- Belirli sorulara cevap verir ve kullanıcılardan yeni sorular öğrenir.
- Veritabanı dosyası üzerinden soru-cevap eşleştirmesi yapar.

## Kullanım Alanları:

- Müşteri hizmetleri sistemlerinde otomatik cevap verme mekanizması olarak kullanılabilir.
- Eğitim materyalleri geliştirme sürecinde öğrenme aracı olarak kullanılabilir.
- Temel yapay zeka uygulamalarının geliştirilmesi için başlangıç noktası olarak kullanılabilir.

## Nasıl Kullanılır?

1. Projeyi bilgisayarınıza klonlayın veya ZIP dosyası olarak indirin.
2. Python ve gerekli kütüphaneleri yükleyin (tkinter, difflib, playsound).
3. Kodu bir Python ortamında çalıştırın.
4. Veritabanı dosyasını belirleyin ve sohbet botunu başlatın.
5. Chat Bot size "Merhaba!" diyecektir. Sohbeti başlatmak için sorularınızı girebilirsiniz.
